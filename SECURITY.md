# Security Policy

## Supported Versions

We support the latest version on the `main` branch.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately and responsibly.

- **Preferred**: GitHub Security Advisories
  - https://github.com/<OWNER>/<REPO>/security/advisories

Please include:
- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any suggested mitigations (if known)

Do **not** open a public issue for security reports.

## Response Timeline

We aim to:
- Acknowledge reports within **72 hours**
- Provide a remediation plan within **14 days**
- Release a fix as soon as feasible

## Responsible Use

NetSec AI Scanner is intended for authorized security testing only. Do not scan systems or networks you do not own or explicitly have permission to test.
